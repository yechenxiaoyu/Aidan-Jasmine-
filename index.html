使用说明

资源库模式说明

  • 本页面通过读取 data 目录下的数据文件来显示内容
  • 每种类型的内容都存储在一个JSON文件中，而不是分散的多个文件
  • 日记数据存储在 data/diaries.json 文件中
  • 纪念日数据存储在 data/memories.json 文件中
  • 相册数据存储在 data/albums.json 文件中
  • 留言数据存储在 data/messages.json 文件中
  • 相册图片存储在 images/ 目录下
  • 要添加新内容，只需要编辑对应的JSON文件并刷新页面

添加新内容的方法

编辑对应的JSON文件，在数组中添加新的对象，然后刷新页面即可看到更新。

日记文件格式示例 
[
  {
    "id": 3,
    "title": "日记标题",
    "date": "2025-07-26",
    "content": "日记内容...",
    "tags": ["标签1", "标签2"]
  }
]
纪念日文件格式示例 
[
  {
    "id": 3,
    "title": "纪念日标题",
    "date": "2025-07-26",
    "description": "纪念日描述..."
  }
]
相册文件格式示例 
[
  {
    "id": 3,
    "title": "相册项标题",
    "date": "2025-07-26",
    "description": "相册项描述...",
    "category": "分类",
    "image": "images/图片文件名.jpg"
  }
]
留言文件格式示例 
[
  {
    "id": 3,
    "content": "留言内容...",
    "author": "留言人",
    "date": "2025-07-26"
  }
]

日记管理

编辑 data/diaries.json 文件来管理日记内容。

  1. 打开 data/diaries.json 文件
  2. 在数组中添加新的日记对象或修改现有内容
  3. 保存文件后点击下面的刷新按钮更新页面内容

纪念日管理

编辑 data/memories.json 文件来管理纪念日内容。

  1. 打开 data/memories.json 文件
  2. 在数组中添加新的纪念日对象或修改现有内容
  3. 保存文件后点击下面的刷新按钮更新页面内容

相册管理

编辑 data/albums.json 文件来管理相册内容。

  1. 打开 data/albums.json 文件
  2. 在数组中添加新的相册对象或修改现有内容
  3. 如果需要添加图片，请将图片文件放入 images/ 目录
  4. 在相册对象中设置 image 字段为图片路径
  5. 保存文件后点击下面的刷新按钮更新页面内容

  2. 在数组中添加新的留言对象或修改现有内容
  3. 保存文件后点击下面的刷新按钮更新页面内容
